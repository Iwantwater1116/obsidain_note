# 切片陣列(Slice)

切片陣列是Golang一種蠻特殊的陣列，它同時有不固定長度與固定長度的特性，Slice有以下的幾種特性與規則：

- Slice擁有<font class="red-text font-bold">長度</font>與<font class="red-text font-bold">容量</font>這兩種特性，這兩個的定義不同
- 長度是不固定的，但容量卻是固定，所以長度怎樣都沒問題，<font class="red-text font-bold">但就是不能大於容量</font>。
- Slice是可以互相copy的，copy後會放在原陣列的後面接續，但不可以超過容量限制。
- Slice也可不限制它的容量跟長度就如圖List一樣，表示法有以下幾種。

```go
//第一種建立法，不表示長度與容量，可以把它當List用
var x []float64

//第二種表示法，只寫了一個參數，代表容量與長度相同。
x := make([]float64,5)

//第三種表示法，後面加了兩個參數，第一個是長度，第二個則是容量。
x := make([]float64,5,10)//建立一個長度5，容量10的Slice

//第四種表示法，直接參考別人的陣列做Slice
arr := [5]float64{1,2,3,4,5}
x := arr[0:5]//參考arr index 0~5的值做參數
```

接下來來看看Slice在Golang的使用方法與基本程式碼吧!

```go
func SlicePractice() {
	//var x []float64 => 第一種建立法

	//x := make([]float64, 5, 10) => 第二種表示法

	/*arr := [5]float64{1,2,3,4,5}
	x := arr[0:5] => 參考別的陣列*/

	slice1 := []int{1, 2, 3}
	slice2 := append(slice1, 4, 5)
	slice3 := []int{8, 8, 7}
	copy(slice2, slice3)
	fmt.Println(slice1, slice2, slice3)
	//copy不能超過容量限制
}
```

Output：

```bash
[1 2 3] [8 8 7 4 5] [8 8 7]
```
