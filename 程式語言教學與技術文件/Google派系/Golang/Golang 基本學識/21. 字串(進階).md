# 字串(進階)

字串也有一些進階的功能，所以另外拿出來說明，直接看程式碼吧

```go
func StringPractice() {
	fmt.Println(
		//true
		strings.Contains("test", "es"),

		//2
		strings.Count("test", "t"),

		//true
		strings.HasPrefix("test", "te"), //用來確認字頭字尾始否有包函某些字串

		//true
		strings.HasSuffix("test", "st"), //用來確認字頭字尾始否有包函某些字串

		//1
		strings.Index("test", "e"),

		//"a-b"
		strings.Join([]string{"a", "b"}, "-"),

		// == "aaaaa"
		strings.Repeat("a", 5),

		//"bbaa"
		strings.Replace("aaaa", "a", "b", 2),

		//[]string{"a","b","c","d","e"}
		strings.Split("a-b-c-d-e", "-"),

		//test
		strings.ToLower("TEST"),

		//"TEST"
		strings.ToUpper("test"),
	)
}
```
